# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview
This project was developed during my internship using Microsoft Power BI. It analyzes customer churn data to identify patterns, trends, and business insights through interactive dashboards.

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

## ✨ Key Features
- Customer Churn Rate Analysis
- Customer Demographics
- Revenue & Subscription Insights
- Interactive Dashboard with Filters
- Data Visualization and Business Insights

## 📁 Repository Contents
- Power BI Dashboard (.pbix)
- Dataset
- Dashboard Screenshots
- Internship Certificate
- Demo Video

## 🎯 Objective
To help businesses understand customer behavior and identify factors contributing to customer churn using interactive visualizations.
